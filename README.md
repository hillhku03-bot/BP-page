# Dota 7.41 英雄环境监测

本项目用于生成并展示 Dota 2 版本 7.41 的英雄环境数据，默认以本地 DB 为主数据源。

## 样本范围

- ESL One Birmingham 2026
- PGL Wallachia 2026 Season 8
- DreamLeague Season 29
- BLAST SLAM VII
- EWC 2026 Regional Qualifiers
- The International 2026 Regional Qualifiers

## 常用命令

```powershell
npm install
npm run build
npm run dev
```
